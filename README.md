# Automatic Curtain Controlled by Gestures

## Project Description

This project implements an automatic curtain controlled by hand gestures.

The system uses:

- Python
- MediaPipe
- OpenCV
- Arduino UNO
- DC Motor FA-130

## Control

👊 Open palm → open curtain

✋ Closed fist → close curtain
